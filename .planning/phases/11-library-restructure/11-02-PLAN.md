       - `year.toString()`
     - Use `.includes()` for substring matching
     - Match any field (OR logic)

  5. Sorting:
     - After search filtering, sort by sortKey/sortDir using the same pattern from CaseCardList.tsx
     - Date sort: compare date_issued strings
     - Company sort: compare company_name strings
     - Provisions sort: compare num_provisions numbers

  6. Pagination:
     - PAGE_SIZE = 20
     - Copy the getPageNumbers helper function from CaseCardList.tsx
     - Copy the Pagination component usage pattern from CaseCardList.tsx

  7. Reset page to 1 on search/sort change (use the filterKey + prevFilterRef + useEffect pattern from CaseCardList.tsx)

  8. Layout — render in order:

     e. **Pagination:** Below case list, same pattern as CaseCardList.tsx
